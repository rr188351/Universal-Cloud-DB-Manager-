# 🌐 Universal Cloud Database Manager by RMAC

A cinematic Python GUI suite for secure admin login, cloud database access, and full-stack data operations — blending visual storytelling with backend precision.
Download it From Release Section

## 🎬 Modules Included

### 🔐 Admin Login Portal
- Credential-based access
- Stylized portal interface with surreal background
- Entry point to SQL and MongoDB modules

### 🧮 iSQL Cloud Manager
- MySQL/Cloud connection via host, port, user, password
- Create/load databases and tables
- Record editor with Insert, Update, Delete, Multi Delete
- Run raw SQL queries
- Logout and database switch options

### 🍃 MongoDB Cloud Manager
- Connect via `mongodb+srv://` or local URI
- Create datasets and collections
- Record editor with full CRUD
- Delete field, collection, or entire database
- Logout and database switch options
